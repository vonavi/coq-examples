Answer

Even-ness isn't actually important to this goal, as it is just a fact about mapping. Either c is the empty list, or c is of the form c = x :: xs = [x] ++ xs so map Nat.even c = Nat.even x ++ map Nat.even xs. As such, you could have a proof like

forall c : list nat,
c = [] \/
(exists (c1 c2 : list nat) (b : bool),
   c = c1 ++ c2 /\
   list_even c1 = [b] /\
   list_even c = b :: list_even c2)


forall c : list nat,
c = [] \/
(exists (c1 c2 : list nat) (b : bool),
   c = c1 ++ c2 /\
   list_even c1 = [b] /\
   list_even c = b :: list_even c2)

  
c: list nat
c = [] \/
(exists (c1 c2 : list nat) (b : bool),
   c = c1 ++ c2 /\
   list_even c1 = [b] /\
   list_even c = b :: list_even c2)

  destruct c as [|x xs];
    [ left; auto
    | right; exists (x :: nil); do 2 eexists; repeat split; eauto ].
Qed.
