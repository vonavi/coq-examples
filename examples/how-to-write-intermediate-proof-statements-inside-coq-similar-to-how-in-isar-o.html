Answer (Li-yao Xia)

You can write assert <lem> to prove an intermediate result <lem> in the middle of a proof. Other variants are assert <lem> by <tactic> to immediately prove <lem> using <tactic>, or assert (<lemname> : <lem>) to give a name to the lemma. Example:

forall n m : nat, n + m = m + n


forall n m : nat, n + m = m + n

  
n, m: nat
n + m = m + n
 
m: nat
0 + m = m + 0
n', m: nat
IH: n' + m = m + n'
S n' + m = m + S n'

  
m: nat
0 + m = m + 0
 
m: nat
m = m + 0

    
m: nat
add_easy_induct_1: forall n : nat, n + 0 = n
m = m + 0

    
m: nat
add_easy_induct_1: forall n : nat, n + 0 = n
m = m
 reflexivity.
  
n', m: nat
IH: n' + m = m + n'
S n' + m = m + S n'
 
n', m: nat
IH: n' + m = m + n'
S (n' + m) = m + S n'

    
n', m: nat
IH: n' + m = m + n'
plus_n_Sm: forall n m : nat, S (n + m) = n + S m
S (n' + m) = m + S n'

    
n', m: nat
IH: n' + m = m + n'
plus_n_Sm: forall n m : nat, S (n + m) = n + S m
S (m + n') = m + S n'
 
n', m: nat
IH: n' + m = m + n'
plus_n_Sm: forall n m : nat, S (n + m) = n + S m
S (m + n') = m + S n'
 
n', m: nat
IH: n' + m = m + n'
plus_n_Sm: forall n m : nat, S (n + m) = n + S m
S (m + n') = S (m + n')
 reflexivity.
Qed.

Documentation on assert: https://coq.inria.fr/distrib/current/refman/proof-engine/tactics.html#coq:tacn.assert

Answer (Pierre Jouvelot)

You can use the have: construct in the ssreflect language of tactics for Coq, with pretty much the same semantics you want, plus a couple of additional nice features related to how this lemma can be used right away (e.g., for rewriting) instead of being given a name.

For a concrete code example, see https://stackoverflow.com/a/71428239/1601580

(* have proof *)
From Coq Require Import ssreflect ssrfun ssrbool.


forall n m : nat, n + m = m + n


forall n m : nat, n + m = m + n

  
n, m: nat
n + m = m + n
 
m: nat
0 + m = m + 0
n, m: nat
IHn: n + m = m + n
S n + m = m + S n

  
m: nat
0 + m = m + 0
 
m: nat
m = m + 0

    
m: nat
(forall n : nat, n + 0 = n) -> m = m + 0

    
m: nat
H: forall n : nat, n + 0 = n
m = m + 0
 
m: nat
H: forall n : nat, n + 0 = n
m = m
 by reflexivity.
  
n, m: nat
IHn: n + m = m + n
S n + m = m + S n
 
n, m: nat
IHn: n + m = m + n
S (n + m) = m + S n
 
n, m: nat
IHn: n + m = m + n
S (m + n) = m + S n

    
n, m: nat
IHn: n + m = m + n
(forall n m : nat, S (n + m) = n + S m) ->
S (m + n) = m + S n

    
n, m: nat
IHn: n + m = m + n
H': forall n m : nat, S (n + m) = n + S m
S (m + n) = m + S n
 
n, m: nat
IHn: n + m = m + n
H': forall n m : nat, S (n + m) = n + S m
m + S n = m + S n
 by reflexivity.
Qed.
